# jest-practice

jestでテストの練習を行うリポジトリ

## stage 1
ドキュメント制覇

- jestをinstall

```
npm i --save-dev jest
```

- ts導入

```
npm i typescript
npm i --save-dev @types/jest

// babel setting
npm i --save-dev @babel/preset-typescript
npm i --save-dev babel-jest @babel/core @babel/preset-env
```
